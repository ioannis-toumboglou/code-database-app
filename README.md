# Java Code Database
<b>Description:</b>

A web application that is connected to a database that contains code files, 
performing CRUD operations. The user is able to search the code with the use
of keywords.
<br><br>
<b>Technologies used:</b>
<br>
<ol>
	<li>Spring MVC</li>
	<li>JSP</li>
	<li>Servlet</li>
	<li>Hibernate</li>
	<li>Apache Tomcat</li>
	<li>MySQL database</li>
</ol>
